# truncatewallet

## Install

Install with [npm](https://www.npmjs.com/):

```sh
$ npm install --save truncatewallet
```

## Usage

Works with strings.

```js
const truncatewallet = require('truncatewallet');

console.log(truncatewallet('0x41a7e9787baB84eAa6f21b6c15dca30FE630684C')); //=> '0x41a7...684C'

```
